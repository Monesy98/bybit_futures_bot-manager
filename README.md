# bybit futures bot manager
Менеджер для торговли с добавлением стратегий + тг ботом
Работает с только с биржей Bybit на фьючерсах

Нстройка:
- В .env Заполнить api ключи и токен бота
- Торговля установлена: изолированная маржа, можно поменять в main А так же расчет свечей при закрытии либо во время изминения. Плечо выбирается в настройках интерфейса.
- Конфигурация стратегий в файле main - timeframe графика, amount - сумма сделки в колличистве монет, ниже увидите настройки стратегий atr - length, multiplier.

На данный момент 2 стратегии. Добавление стратегий на python в папку strategies, добавляем импорт в main и поле в index.html

Процесс установки:
- Требуется python3.11
- Создаем папку mkdir /opt/bybit_futures_bot/
- Установите python3-venv, если не установлен
- sudo apt update && sudo apt install python3-venv -y
- Переходим в папку проекта
- cd /opt/bybit_futures_bot/
- Создайте виртуальное окружение в папке проекта
- python3 -m venv venv
- Активируйте его
- source venv/bin/activate
- Установим зависимости
- pip install -r requirements.txt
- Запускаем python main.py

Веб интерфейс на http://localhost:8000
Можно управлять через тг бота + он отправляет уведомления о позициях в боте введите /help для отображения комманд.

При запуске стратегии бот синхранихируется с открытой сделкой если есть. При остановке через ctrl+C Либо через интерфейс или бота. Открытые позиции которые есть в бд бот закрывает автоматический.
