# bybit futures bot manager
Менеджер для торговли с добавлением стратегий + тг ботом
Работает только с биржей Bybit на фьючерсах

# Настройка:
- В .env Заполнить api ключи и токен бота
- Торговля: установленна изолированная маржа, можно поменять в main, расчет стратегии при закрытии пред, свечи выбирайте 5 мин. Плечо выбирается в настройках интерфейса.
- Конфигурация стратегий в файле main - timeframe графика, amount - сумма сделки в колличистве монет, ниже увидите настройки стратегий atr - length, multiplier.

На данный момент 2 стратегии. Добавление стратегий на python в папку strategies, добавляем импорт в main и поле в index.html

# Процесс установки:
- Требуется python3.12
- Создаем папку mkdir /opt/bybit_futures_bot/
- Установите python3-venv, если не установлен
- sudo apt update && sudo apt install python3-venv -y
- Переходим в папку проекта
- cd /opt/bybit_futures_bot/
- Создайте виртуальное окружение в папке проекта
- python3 -m venv venv
- Активируйте его
- source venv/bin/activate
- Установим зависимости
- pip install -r requirements.txt
- Запускаем python main.py

Веб интерфейс для управления http://localhost:8000
Можно так же управлять через тг бота + он отправляет уведомления о позициях в боте введите /help для отображения комманд.

При запуске стратегии бот синхранихируется с открытой сделкой если есть. При остановке через ctrl+c Либо веб интерфейс или бота, открытые позиции бот закрывает автоматический.
